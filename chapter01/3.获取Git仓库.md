## 获取Git仓库



### 单词

+ `clone`



### 语法

```
git clone <address>
```



### 语义

#### 克隆一个仓库

+ 最简洁指令

```
git clone https://gitee.com/leobod/GitGuide.git
或者也可以使用 git:// 协议或者使用 SSH 传输协议
git clone git@gitee.com:leobod/GitGuide.git
```

+ 克隆并指定文件名

```
git clone https://gitee.com/leobod/GitGuide.git GGuide
```
